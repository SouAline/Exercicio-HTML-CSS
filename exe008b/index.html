Para que você consiga ter um destaque quando estiver falando sobre código fonte basta usar a tag "code": Vamos a um exemplo: A sintaxe in(input()) é muito usada na linguagem Pyhton, como podemos observar os espaçamentos entre as letras digitadas dentro desta tag permenaceram iguais permitindo uma melhor visualização do conteúdo descrito

Agora se quiseres adicionar um trecho da linguagem Python a formatação necessita ser diferente, usaremos o code dentro de outra tag a "pre".

        
    num = int(input('Digite um numero: '))
    If num % 2 == 0:
        print(f'Este número {num} é par))
    Else:
        print (f'Este numero {num} é ímpar')

A tag "pre" considera todos os espaços usados, tanto na horizontal quanto na vertical, cuidado para não sair do uso semantico. Vou acresentar algo aqui para praticar o texto riscado hahahahaha.
