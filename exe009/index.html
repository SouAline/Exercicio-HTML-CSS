Listas Ordenadas

Para criar listas ordenadas você percisa colocar os itens envelopadas na tag "li" dentro de outra tag chamada "ol" Segue exemplo:

  1. Acordar
  2. Escovar os dentes
  3. Tomar café
  4. Lavar roupas
  5. Estudar
  6. Fazer o almoço

Listas não ordenadas

Para criar listas não ordenadas você precisa colocar os itens envelopadas na tag "li" dentro de outra tag "ul". Segue exemplo:

  • Maça
  • Arroz
  • Feijão
  • Batata
  • Couve
